# HotelRevAI Project

HotelRevAI is an AI-driven revenue analysis project developed during Infosys Internship.
It focuses on analyzing hotel booking data to generate useful business insights.
The project uses Power BI for data visualization and reporting.
Data cleaning and preprocessing were done using Power Query.
A star schema data model was created for efficient analysis.
Key KPIs like Revenue, Bookings, ADR, and RevPAR were calculated.
DAX was used to create measures and calculations.
Interactive dashboards were designed to show trends and patterns.
The project helps improve hotel decision-making and profitability.
Overall, it enhanced skills in data analytics and business intelligence.
